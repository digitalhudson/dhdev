Welcome to Digital Hudson!

Digital Hudson is a project created by Tim Moore to achieve certain goals.

You can learn more about this project from Tim and one day it promises to make your job more easy.

Digital Hudson is a web application based on:

  • Scala, aka Master's level Java, as the programming language
  • The infamous SBT build tool
  • Play Framework for a web application framework
  • Play2-auth as authentication module for Play
  • PostgreSQL, the open source world's greatest database
  • Slick as a functional relational database mapper for the access layer
  • Slick-pg extends Slick to support PostgreSQL's etended data types
  • Foundation as a front-end framework
  • Git and Github for version control
  • SalesForce's Heroku for PaaS functionality

Check back here to see progress over the next couple of months.
